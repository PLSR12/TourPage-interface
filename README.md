## :house: About the project
Aplicação completa de uma Agência de Turismo.

Para a criação da interface foi utilizado: React, styled-components, react-toastify para a criação dos toasts de sucesso ou erro, react-router-dom, axios, e a biblioteca yup para fazer a validação dos dados.

Na parte da tela do administrador , foi utilizado a biblioteca MUI para o uso dos ícones e criação das tabelas

In the project directory, you can run:

Runs the app in the development mode.\
